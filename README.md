# 🦴 Ramalan Tulang Cina (称骨算命)

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-green.svg)](https://m-sugiarto.github.io/ramalan-tulang-cina/)

> *Tool kalkulator ramalan tulang Cina (Bone Weight Fortune) berdasarkan metode kuno 袁天罡 (Yuan Tiangang) dari Dinasti Tang.*

## 💭 Cerita di Balik Tool Ini

Semua bermula dari obrolan santai bersama seorang teman yang sangat tertarik dengan ramalan tulang Cina (称骨算命). Dia bercerita dengan antusias bagaimana metode kuno dari Dinasti Tang ini menghitung "berat tulang" seseorang berdasarkan tanggal dan waktu lahir, lalu memberikan ramalan yang konon bisa menggambarkan perjalanan hidup seseorang.

Awalnya saya hanya mendengarkan dengan penasaran. Namun semakin dalam saya pelajari, semakin menarik metode ini — bukan hanya dari sisi ramalannya, tapi juga dari sisi budaya, matematika, dan filosofi Tiongkok kuno di baliknya.

Dari situlah muncul ide untuk membuat tool ini. **Bukan untuk dipercaya secara serius**, tapi lebih sebagai:

- 🎭 **Hiburan** - Sekadar iseng melihat "berapa berat tulang" kita menurut metode kuno
- 📚 **Edukasi** - Melestarikan pengetahuan budaya Timur yang menarik ini
- 🔬 **Eksperimen** - Menerapkan konversi kalender lunar yang akurat dalam web modern

Jadi silakan coba, nikmati hasilnya, dan **jangan terlalu serius** — anggap saja sebagai hiburan yang menyenangkan! 😄

## ✨ Fitur

- ✅ Konversi tanggal Masehi ke Lunar **akurat 100%** (library `lunar-javascript`)
- ✅ Perhitungan berat tulang berdasarkan Tahun, Bulan, Hari, Jam (Shichen)
- ✅ Teks asli 称骨歌 dalam **Hanzi + Pinyin + Terjemahan Indonesia**
- ✅ Data ramalan lengkap dari 2.1 Liang sampai 7.2 Liang
- ✅ Responsive design (mobile & desktop)
- ✅ 100% client-side, tidak perlu server
- ✅ Running text disclaimer untuk transparansi

## 📦 Deploy ke GitHub Pages

1. Fork repository ini
2. Buka **Settings** → **Pages**
3. Pilih branch `main` dan folder `/ (root)`
4. Website akan live di `https://[username].github.io/ramalan-tulang-cina/`

## 📖 Cara Penggunaan

1. **Masukkan tanggal lahir Masehi** dan **jam lahir** (format 24 jam)
2. Klik **"🔄 Konversi ke Kalender Lunar"** untuk konversi otomatis
3. Klik **"✨ Isi Otomatis ke Form"** untuk mengisi form
4. Klik **"🔮 Hitung Ramalan Tulang"** untuk melihat hasil
5. Lihat hasil berupa:
   - Berat tulang dalam satuan Liang
   - Teks asli Hanzi + Pinyin
   - Terjemahan Bahasa Indonesia
